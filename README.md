# 🎤 InterviewCoach AI

![Python](https://img.shields.io/badge/Python-3.10+-blue) ![Streamlit](https://img.shields.io/badge/Streamlit-1.32+-red)

**AI-powered mock interview coach.** Select a role, answer real interview questions, and get instant feedback on your answer quality.

## Features
- 🎯 Role-specific questions: SWE, Data Science, ML, PM, HR
- 🎤 Text answer input (voice coming soon)
- 📊 Score on: Clarity, Technical Depth, Communication, Confidence, Relevance
- 💬 Follow-up questions from AI
- 📝 Full session transcript + summary
- 🏆 Final readiness score

## Run
```bash
pip install -r requirements.txt
GEMINI_API_KEY=your_key streamlit run app.py
```
